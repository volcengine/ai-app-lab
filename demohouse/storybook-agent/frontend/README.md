# 故事书生成应用前端

## 项目介绍

本前端应用是故事书生成系统的用户界面，提供直观的交互方式让用户生成故事书和连环画内容。用户可以通过简单的提示词，系统会自动生成完整的故事内容和配套插图，支持故事书和连环画两种生成模式。

## 主要功能

- 故事主题输入：用户可以输入故事主题或提示词
- 参考图片上传：支持上传参考图片来影响生成风格
- 生成模式选择：支持故事书 (storybook) 和连环画 (comics) 两种模式
- 图片尺寸设置：可自定义生成图片的尺寸
- 生成内容预览：直观展示生成的故事内容和图片

## 技术栈

- React + TypeScript
- Modern.js
- Tailwind CSS

## 环境要求

- Node.js 22+（推荐使用项目指定的版本，详见 .nvmrc）
- pnpm 9+ 包管理器

## 开发流程

### 安装依赖

```bash
pnpm install
```

### 启动开发服务器

```bash
pnpm dev
```

服务启动后，可通过浏览器访问前端应用。

### 构建生产版本

```bash
pnpm build
```

### 本地预览生产构建

```bash
pnpm serve
```

## 注意事项

- 确保后端服务已正常启动，前端应用需要与后端API进行交互
- 生成内容需要使用ARK API，确保后端已配置相关API密钥
